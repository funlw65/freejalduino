freejalduinom3.jal Library

16 Apr, 2010 | Category: Tutorials | Author: Vasi

This library is used for FreeJALduino Mega 3c boards, based on 18F4550 and Arduino hardware design. It just redefine the 18F2550 pinout, in concordance with Arduino board notation. Can be found in

C:\jal\lib

I advise you to look at 18f4550.jal source to see the real pinout, registers and peripherals definitions

-- This library is made to help Arduino Language programmers to migrate
-- to JAL language and Jallib library using FreeJALduino MEGA v3 board
-- Is compatible backward
-- with FreeJALduino v.4 board (28 pin, 18F2550)
include 18f4550 -- target PICmicro
-- even though the external crystal is 20 MHz, the configuration is such that
-- the CPU clock is derived from the 96 Mhz PLL clock (div2), therefore set
-- target frequency to 48 MHz
pragma target clock 48000000  -- oscillator frequency
-- Now we tell to compiler that the application use bootloader and must start at 0x0800
pragma bootloader loader18
-- And that this application does not use fuses (which are defined already in bootloader)
pragma fuses no

-- Digital I/O procedures

-- Setting a digital pin as input or output
-- You also can set an analog pin as digital output
-- Example:
-- pinMode(D21_direction, OUTPUT)
procedure pinMode(bit in out x, bit in y) is
x = y
end procedure

-- Writting 0 or 1 to a digital pin
-- Example:
-- pinMode(D21_direction, OUTPUT)
-- digitalWrite(D21, LOW) -- outputing 0 to Digital 21
procedure digitalWrite(bit in out x, bit in y) is
x = y
end procedure

-- Reading a digital pin value
-- Example:
-- var byte pin_value
-- setMode(D22_direction, INPUT)
-- pin_value = digitalRead(D22)
function digitalRead(bit in x) return bit is
var bit i
i = x
return i
end function

-- Analog Input functions
-- This function is compatible with Arduino Language
-- Anyway, you have a better alternative with original jallib libraries
-- Example:
-- var word pot_value
-- setMode(A0_direction, INPUT)
-- pot_value = analogRead(0)
function analogRead(byte in channel) return word is
var word result = 0
ADCON0 = channel << 2
ADCON1 = 14 - channel
ADCON2 = 0xBE
ADCON0_ADON = 1
for 9 loop
  assembler
    NOP
  end assembler
end loop
ADCON0_GO = 1
while (ADCON0_GO) loop
end loop
result = ADRES
ADCON0_ADON = 0
return result
end function

-- Math functions
function min(dword in x, dword in y) return dword is
if x < y then
  return x
else
  return y
end if
end function

function max(dword in x, dword in y) return dword is
if x > y then
  return x
else
  return y
end if
end function

function constrain(dword in z, dword in x, dword in y) return dword is
if z < x then
  return x
elsif z > y then
  return y
else
  return z
end if
end function
